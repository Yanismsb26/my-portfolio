Design Web Moderne

Description du projet

Ce projet consistait à concevoir une interface utilisateur moderne et responsive pour un site e-commerce spécialisé dans les produits artisanaux. L'objectif était de créer une expérience utilisateur fluide et intuitive tout en mettant en valeur les produits.

Technologies utilisées

  • HTML5, CSS3, JavaScript
  • Bootstrap 5
  • Adobe Photoshop
  • Adobe XD pour le prototypage

Résultats

Le design final a permis une augmentation de 35% du taux de conversion et une réduction de 20% du taux de rebond.

Application Mobile

Description du projet

Technologies utilisées

  • React Native
  • Node.js
  • MongoDB
  • AWS S3

Résultats

Campagne Publicitaire

Description du projet

Conception et mise en œuvre d'une campagne publicitaire complète pour le lancement d'une nouvelle marque de vêtements éco-responsables. La campagne incluait des supports print, digital et des réseaux sociaux.

Technologies utilisées

  • Adobe Creative Suite
  • Analytics
  • Stratégie de contenu
  • Planification média

Résultats

La campagne a généré plus de 200 000 impressions et a contribué à atteindre 150% de l'objectif de ventes initial pour le premier trimestre.
